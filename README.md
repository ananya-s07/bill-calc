# Energy Consumption Calculator (C++)

A menu-driven C++ application to calculate household energy consumption and estimate electricity bills using real-world electrical formulas.

## Features
- Add appliances with power rating and daily usage
- Calculate appliance-wise and total energy consumption (kWh)
- Monthly electricity bill estimation
- Persistent storage using file handling
- Object-Oriented Design

## Formula Used
Energy (kWh) = (Power × Hours × Days) / 1000

## Technologies
- C++
- OOP Concepts
- File Handling

## How to Run
1. Compile using:
   g++ main.cpp Appliance.cpp -o energy
2. Run:
   ./energy
